innerHTML / innerText를 이용한 내용 변경

자바스크립트에서 요소의 내용을 읽거나 변경할 때 innerText라는 속성을 사용한다.(태그인식 X , 태그가 적용되지 않고문자열로 나타냄)

요소의 내용을 변경할 때 내용에 포함된 태그를 실제 태그로 인식시키기 위해서는 innerHTML을 사용해야한다. (태그인식 O ,태그가 적용된다. )

window.confirm()을 이용한 데이터 입력

    어떤 질문에 "예 / 아니오"의 결과를 얻고자 할 때 사용한다.
    
    경고창에 "확인" , "취소" 버튼이 나타나며
    "확인" 클릭 시 true / "취소" 클릭 시 false가 반환된다.

window.prompt()를 이용한 데이터 입력

    텍스트를 적을 수 있는 필드와 확인 / 취소 버튼이 있는 대화 상자를 출력

    텍스트를 입력한 후 확인버튼을 누르면 입력된 텍스트가 반환된다.
