문자열과 숫자의 + 연산

    
    문자열 + 숫자 = 문자열 (이어쓰기)
    숫자 + 문자열 = 문자열 (이어쓰기)
    숫자 + 숫자 + 문자열 = 숫자가 먼저 계산된 후 문자열로 이어써짐 ex) 1 + 1 + 문 = 2문
                            (사칙연산 우선 순위가 적용된다.)

문자열 -> 숫자

    input 태그 등을 이용한 입력된 숫자는 숫자가 아닌 문자열로 인식된다.

    자바스크립트에서 문자열 -> 숫자로 바꿀수 있는 
    내장객체 Number()
    내장함수 parseInt(), parseFloat()가 있다.
