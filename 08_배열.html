배열

배열이란?

    자바스크립트에서는 리터럴 값이 지정되기 전 까지 
    변수의 자료형이 지정되지 않으므로
    배열 선언 시 자료형이 지정되지 않아 모든 자료형을 보관할 수 있다.
    == 자바의 컬렉션(List)과 비슷하다.

    - 모든 자료형 저장가능
    - 배열 길이 제한이 없다.

배열 관련 함수

    자바스크립트에서는 배열도 객체이기 때문에 함수(Method)를 가지고 있다.
    
    indexOf() : 일치하는 배열 요소의 인덱스를 반환

    concat() : 다수의 배열을 결함

    toString() , join()  : 배열 요소를 결합하여 하나의 문자열로 반환
        - toString() : 요소 사이에 , 를 찍음
        - join() : 요소 사이에 구분자를 지정
    
    sort() : 배열을 오름차순 정렬
    reverse() : 배열의 순서를 뒤집는다 ( sort()로 오름차순 후 선언 )

indexOf() : 일치하는 배열 요소의 인덱스를 반환

"사과","딸기","바나나","파인애플","토마토"

concat() : 다수의 배열을 결함

toString() , join() : 배열 요소를 결합하여 하나의 문자열로 반환

    - toString() : 요소 사이에 , 를 찍음
    - join() : 요소 사이에 구분자를 지정

sort() : 배열을 오름차순 정렬
reverse() : 배열의 순서를 뒤집는다 ( sort()로 오름차순 후 선언 )

push() : 배열의 맨 뒤에 요소 추가
pop() : 배열의 맨 뒤 요소를 꺼내옴(배열에서 제거) shift() : 배열의 맨 앞에 요소를 제거 unshift() : 배열의 맨 앞에 요소를 추가

slice() : 배열 요소를 선택해서 자르기
splice() : 배열에 지정한 index 위치 요소 제거 후 새로운 요소를 추가
